Missing module

When compiler says that module is missing - which package you need to install. 
Css          - elm install rtfeldman/elm-css
Html.Styled  - > Css
Http         - elm install elm/http
Json         - elm install elm/json
Json.Decode  - > Json
Svg          - elm install elm/svg
Svg.Styled   - > Css
Time         - elm install elm/time
Url.Parser   - elm install elm/url
